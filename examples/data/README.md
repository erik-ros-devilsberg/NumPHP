# Example data fixtures

`iris.csv` — 15 rows from the UCI Iris dataset (Fisher 1936), 4 numeric columns
(`sepal_length, sepal_width, petal_length, petal_width`), 1 header row. 5 rows
each from setosa, versicolor, and virginica. Public domain. Source:
<https://archive.ics.uci.edu/dataset/53/iris>.

Class labels are intentionally omitted — the example treats this as a numeric
analysis pipeline, not a classification problem.
